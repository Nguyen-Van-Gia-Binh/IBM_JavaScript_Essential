A simple and functional todo list application for task management.

## Features
- Add new tasks
- Mark tasks as complete
- Delete individual tasks
- Clear completed tasks
- Clear all tasks

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)

## How to Use
1. Open `todoList.html` in a browser
2. Type a task in the input field
3. Click "Add" or press Enter
4. Click on a task to mark as complete
5. Use "Clear Done" to remove completed tasks
6. Use "Clear All" to remove everything

## Concepts Demonstrated
- Event listeners
- DOM manipulation
- Array management
- Task state toggling
