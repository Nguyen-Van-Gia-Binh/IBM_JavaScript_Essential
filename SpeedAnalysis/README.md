# ⌨️ Speed Typing Analysis

A typing speed test application to measure words per minute (WPM) and accuracy.

## Features
- Start/End test controls
- Display sample text to type
- Calculate typing speed (WPM)
- Measure accuracy percentage
- Real-time comparison

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)

## How to Use
1. Open `speedAnalysis.html` in a browser
2. Click "Start Test" to begin
3. Type the displayed text in your input area
4. Click "End Test" to see results
5. View your WPM and accuracy

## Concepts Demonstrated
- Timer functions (`Date.now()`)
- String comparison
- Performance calculations
- User input handling
