Understanding JavaScript Variable Declarations

var

  • Function-scoped
  • Can be redeclared
  • Hoisted (initialized as undefined)
var x = 10;
var x = 20; // OK

let

  • Block-scoped
  • Cannot be redeclared
  • Hoisted but not initialized (TDZ)
let y = 10;
y = 20; // OK
// let y = 30; // Error!

const

  • Block-scoped
  • Must be initialized
  • Cannot be reassigned
const z = 10;
// z = 20; // Error!
// Must use initial value
