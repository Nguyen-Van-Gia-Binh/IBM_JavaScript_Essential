A weather report application that fetches real-time weather data from OpenWeather API.

## Features
- Enter city name to get weather
- Display temperature, humidity, and conditions
- Uses OpenWeather API

## Technologies
- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- OpenWeather API

## How to Use
1. Open `weatherReport.html` in a browser
2. Get a free API key from [OpenWeather](https://openweathermap.org/api)
3. Enter your API key in the input field
4. Type a city name (e.g., "Hanoi", "London")
5. Click "Get Weather" to see results

## API Setup
1. Visit https://openweathermap.org/api
2. Create a free account
3. Generate an API key
4. Use the key in the application
