A product feedback portal for a luxury brand, demonstrating form handling in JavaScript.

## Features
- Collect user information (name, age, email, job, designation)
- Product type selection (Perfume, Cream, Oils)
- Feedback and experience text areas

## Technologies
- HTML5
- CSS3

## How to Use
1. Open `formHandling.html` in a browser
2. Fill in all required fields
3. Select product type from dropdown
4. Enter your feedback and experience
5. Click "Submit Feedback" to see summary

## Concepts Demonstrated
- Form validation
- DOM manipulation
- Event handling
- User feedback display
